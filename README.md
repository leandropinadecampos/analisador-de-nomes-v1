# 🧠 Analisador de Nomes v1
Projeto simples em Python para análise de nomes (estudo de strings).

## 📌 Funcionalidades
- Separar primeiro nome
- Identificar nome do meio
- Identificar último nome
- Contar quantidade de letras (sem espaços)

## 🚀 Tecnologias utilizadas
- Python

## 🎯 Objetivo
Este projeto foi desenvolvido com o objetivo de praticar:
- Manipulação de strings
- Uso de listas
- Entrada e saída de dados

## 🔄 Futuras melhorias
- Ignorar palavras como "da", "de", "do"
- Contar quantidade de nomes
- Melhorar validações
- Criar uma interface mais interativa


## ▶️ Como executar
1. Abra o arquivo no Python
2. Execute o programa
3. Digite seu nome completo quando solicitado
